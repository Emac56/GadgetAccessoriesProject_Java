# EMAC'S GADGET SHOP (Java Console Application)

A simple Java console-based ordering system for gadget and accessories.  
Created as a final output and practice project to apply Java fundamentals.

---

## Features

- Order gadget accessories
- Buy more items in one transaction (Yes / No option)
- Multiple items per order
- Auto-generated receipt
- Automatic VAT computation (12%)
- View recent order

---

## Technologies Used

- Java (JDK 8+)
- Scanner
- Arrays and ArrayList
- Console Application
